# Открытка «Вы скоро станете бабушкой и дедушкой» 💌

Сайт-открытка для объявления о беременности. Чистый HTML/CSS/JS — без сборки,
можно открыть `index.html` локально или задеплоить на GitHub Pages / Vercel.

## Сценарий

1. **Загрузка** — фото папы и мамы выезжают слева и справа к центру, между ними появляется сердечко.
2. **Скролл вниз** — текст «У нас есть для вас одна очень важная новость…» и кнопка **«Узнать новость ✨»**.
3. **Кнопка** — включается видео с аистом (`assets/stork.mp4`). Пока видео не загружено, показывается временная анимация: аист пролетает, роняет корзинку с малышом, мама и папа её ловят.
4. **Ослепляющая вспышка** — переход к финалу.
5. **Финал** — снимок УЗИ, текст «Вы скоро станете бабушкой и дедушкой!» и салют на заднем фоне.

## Как заменить временные материалы

Просто положите свои файлы в папку `assets/` с такими именами — код подхватит их автоматически:

| Файл | Что это | Рекомендации |
|---|---|---|
| `assets/papa.jpg` | Фото папы | вертикальное, ~3:4 |
| `assets/mama.jpg` | Фото мамы | вертикальное, ~3:4 |
| `assets/uzi.jpg` | Снимок УЗИ | горизонтальный, ~4:3 |
| `assets/stork.mp4` | Видео с аистом | mp4 (H.264); после окончания видео автоматически идёт переход к финалу |

Пока этих файлов нет, отображаются SVG-заглушки (`placeholder-*.svg`) и анимация аиста вместо видео. Ничего в коде менять не нужно.

## Тексты
